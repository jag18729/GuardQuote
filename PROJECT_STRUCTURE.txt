API Endpoints (Backend runs on :3000):
====================================

POST   /auth/register                 Register new user
POST   /auth/login                    Login, get JWT token

GET    /users                         Get all users (protected)

GET    /quotes                        Get my quotes (protected)
POST   /quotes                        Create quote (protected)
GET    /quotes/:id                    Get quote by ID (protected)
PATCH  /quotes/:id                    Update quote (protected)
DELETE /quotes/:id                    Delete quote (protected)
